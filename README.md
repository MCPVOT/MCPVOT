## ⚡ x402 Facilitator — Gasless VOT Payments

The **x402 V2 Facilitator Machine** enables gasless micropayments using the **$VOT token** on Base network.

| Component | Detail |
|-----------|--------|
| **Token** | $VOT on Base |
| **Contract** | `0xc1e1E7aDfDf1553b339D8046704e8e37E2CA9B07` |
| **Protocol** | x402 HTTP 402 Payment Required |
| **Network** | Base (Ethereum L2) |
| **Facilitator** | Gasless — users pay zero gas fees |

### x402 How It Works

```
┌──────────┐    HTTP 402     ┌──────────────┐    Verify    ┌──────────┐
│  Client   │ ──────────────► │   x402       │ ──────────► │  $VOT    │
│  (Agent)  │ ◄────────────── │  Facilitator │ ◄────────── │  Contract│
└──────────┘    200 OK +      └──────────────┘    Valid     └──────────┘
                Data            (Gasless)          Payment
```

1. **Agent requests service** — HTTP call with payment header
2. **Facilitator verifies** — checks VOT balance & approval
3. **Gasless transfer** — facilitator sponsors the gas
4. **Service delivered** — agent gets response + payment settled
