### 🥑 Active Project — [Pequi](https://pequi-lyart.vercel.app)

AI-powered real estate agent specialized in **Ibagué, Tolima, Colombia**.

| Feature | Stack |
|---------|-------|
| Ley 820 compliant contracts | Next.js 15, React 19 |
| Neighborhood pricing (2026) | Prisma 6, PostgreSQL (Neon) |
| Property search & AI chat | Z.AI GLM-5, OpenRouter |
| Digital signatures | Clerk Auth, Vercel Edge |
| Cyberpunk V7 UI | Tailwind CSS 4, Pixel Fonts |
